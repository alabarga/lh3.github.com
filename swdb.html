Software and Database

Ongoing Projects

  • SAMtools: SAM (Sequence Alignment/Map) format is a generic alignment format for large-scale nucleotide alignment. SAMtools is a set of utilities that process alignments in the SAM format. I am one of the main contributors to the SAM specification and the developer of C version of SAMtools.
  • BWA: BWA is a new read alignment program based on Burrows-Wheeler Transform (BWT). It short-read alignment mode is an order of magnitude faster than MAQ while achieving similar alignment accuracy. The long-read mode is several times faster than SSAHA2 and BLAT. I am the developer.
  • MAQ (Map and Assembly with Qualities): MAQ is a set of programs that maps short reads to the reference and assembles the consensus from the alignment. I am the chief developer.
  • Treesoft: TreeSoft provides several softwares for reconstructing or manipulating phylogenetic trees. I wrote NJTREE and initial codes for TreeFam.
  • TreeFam (Tree families databases): TreeFam is a database of phylogenetic trees of animal genes. I am one of the key founders of this project.

Historical Projects

  • Trace Utilities: Trace-utils is a program that recovers heterozygotes from trace files. This program works and can produce reasonable results. Some of its original features are still missing in similar softwares developed recently. Trace-utils was written in 2004 but has not been really finished.

Involved Projects

  • PigGIS (Pig Genomics Informatics System): PigGIS is a pig genome database based on 0.66X WGS reads, 0.7 million ESTs and 1 million GenBank records. Jue Ruan implemented the whole database, Yiran Guo wrote the paper and I designed the strategy.
  • SNAP (SNP Annotation Platform): SNAP integrates various information from several databases. SNAP was developed by Shengting Li, Lijia Ma and Soren Vang. I designed the basic strategy.
  • CAT (Cross-species Alignment Tool): CAT is able cDNAs to the genome of a different species. Intra-species cDNA-to-genome alignment of course works, too. XAT was written by Liang Guan. I designed the strategy, organized the paper and wrote a small part of the codes.
